# Maluco da IA 👽🍀

Assistente de IA interno para equipes da Zazz Internet.

## Sobre o projeto

Bot de WhatsApp com IA (Claude Haiku) integrado ao N8N, Evolution API e dashboard web para gerenciamento.

## Funcionalidades

- Responde mensagens de texto e áudio (transcrição via Whisper)
- Cria tarefas no Notion automaticamente
- Lê POPs e procedimentos da empresa
- Aprende novas regras via WhatsApp ("Claude aprenda: ...")
- Relatórios diários, semanais e mensais do grupo
- Bom dia automático nos dias úteis
- Dashboard web para gerenciamento completo

## Stack

| Componente | Tecnologia |
|-----------|-----------|
| Automação | N8N |
| WhatsApp | Evolution API v2 |
| IA | Claude Haiku (Anthropic) |
| Transcrição | OpenAI Whisper |
| Banco de dados | PostgreSQL |
| Tarefas | Notion API |
| Dashboard | Next.js 14 + Tailwind CSS |
| Hospedagem | CloudFy / Railway |

## Dashboard

Acesse: https://maluco-dashboard-production.up.railway.app

### Funcionalidades do dashboard

- **Visão Geral**: status dos bots, mensagens do dia, erros
- **Treinamento**: gerenciar regras de comportamento da IA
- **POPs**: procedimentos operacionais consultados pelo bot
- **System Prompt**: instruções base do assistente
- **Admin**: gerenciar filiais, colaboradores e configurações
