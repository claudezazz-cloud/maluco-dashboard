### Como funciona o deploy automático

1. Você edita o `workflow.json` no repositório
2. O GitHub Action dispara automaticamente
3. Ele injeta as chaves de API (que ficam como placeholders no arquivo)
4. Faz POST para o webhook do N8N no CloudFy
5. O N8N atualiza o workflow sozinho

**Você NÃO precisa mexer nas credenciais nem nas chaves de API.** Elas ficam salvas como GitHub Secrets e são injetadas automaticamente no deploy.

---

### Regras ao editar o workflow.json

1. **Obtenha o SHA atual** do arquivo antes de qualquer atualização (obrigatório pela API do GitHub)
2. **NÃO substitua** os placeholders de secrets — deixe exatamente assim:
   - Chave Anthropic: `YOUR_ANTHROPIC_API_KEY`
   - Token Notion: `YOUR_NOTION_TOKEN`
3. **Faça PUT** no arquivo via API do GitHub com o conteúdo atualizado + SHA atual
4. O deploy é automático após o push
